# Blinkit Sales & Operations Dashboard – Power BI

This project presents an interactive Power BI dashboard analyzing Blinkit-style quick commerce sales and outlet performance.

## Key Insights
- $1.2M+ total sales analyzed
- Outlet-wise performance by size and location tier
- Category-level sales contribution
- Average sales and customer ratings tracking
- Comparison of grocery stores vs supermarket formats

## Tools Used
- Power BI
- DAX
- Data Cleaning & Modeling
